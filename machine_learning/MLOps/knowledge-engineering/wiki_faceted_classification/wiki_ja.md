A faceted classification is a classification scheme used in organizing knowledge into a systematic order. 
ファセット分類は、知識を体系的に整理するために使用される分類スキームです。

A faceted classification uses semantic categories, either general or subject-specific, that are combined to create the full classification entry. 
ファセット分類は、一般的または特定の主題に関する意味的カテゴリーを使用し、それらを組み合わせて完全な分類項目を作成します。

Many library classification systems use a combination of a fixed, enumerative taxonomy of concepts with subordinate facets that further refine the topic. 
多くの図書館分類システムは、概念の固定された列挙的分類法と、トピックをさらに洗練させる従属ファセットの組み合わせを使用しています。



## Definition 定義

There are two primary types of classification used for information organization: enumerative and faceted. 
情報の整理に使用される主な分類には、列挙型とファセット型の2つがあります。

An enumerative classification contains a full set of entries for all concepts. 
列挙型分類は、すべての概念に対する完全なエントリのセットを含みます。

A faceted classification system uses a set of semantically cohesive categories that are combined as needed to create an expression of a concept. 
ファセット型分類システムは、概念の表現を作成するために必要に応じて組み合わせられる意味的に一貫したカテゴリのセットを使用します。

In this way, the faceted classification is not limited to already defined concepts. 
このように、ファセット型分類は既に定義された概念に制限されません。

While this makes the classification quite flexible, it also makes the resulting expression of topics complex. 
これにより、分類は非常に柔軟になりますが、結果として得られるトピックの表現は複雑になります。

To the extent possible, facets represent "clearly defined, mutually exclusive, and collectively exhaustive aspects of a subject. 
可能な限り、ファセットは「明確に定義され、相互に排他的で、集合的に包括的な主題の側面」を表します。

The premise is that any subject or class can be analyzed into its component parts (i.e., its aspects, properties, or characteristics)." 
前提は、任意の主題またはクラスはその構成要素（すなわち、その側面、特性、または特徴）に分析できるということです。

Some commonly used general-purpose facets are time, place, and form. 
一般的に使用されるファセットには、時間、場所、形態などがあります。

There are few purely faceted classifications; the best known of these is the Colon Classification of Shiyali Ramamrita Ranganathan, a general knowledge classification for libraries. 
純粋なファセット型分類はほとんど存在せず、最もよく知られているのは、図書館の一般知識分類であるShiyali Ramamrita RanganathanのColon Classificationです。

Some other faceted classifications are specific to special topics, such as the Art and Architecture Thesaurus and the faceted classification of occupational safety and health topics created by D. J. Foskett for the International Labour Organization. 
他のファセット型分類は、Art and Architecture Thesaurusや、国際労働機関のためにD. J. Foskettが作成した職業安全と健康に関するトピックのファセット型分類など、特定のトピックに特化しています。

Many library classifications combine the enumerative and faceted classification techniques. 
多くの図書館の分類は、列挙型とファセット型の分類技術を組み合わせています。

The Dewey Decimal Classification, the Library of Congress Classification, and the Universal Decimal Classification all make use of facets at various points in their enumerated classification schedules. 
デューイ十進分類法、国立公文書館分類法、ユニバーサル十進分類法は、すべての列挙された分類スケジュールのさまざまなポイントでファセットを利用しています。

The allowed facets vary based on the subject area of the classification. 
許可されるファセットは、分類の主題領域に基づいて異なります。

These facets are recorded as tables that represent recurring types of subdivisions within subject areas. 
これらのファセットは、主題領域内の繰り返し出現するタイプの細分化を表すテーブルとして記録されます。

There are general facets that can be used wherever appropriate, such as geographic subdivisions of the topic. 
トピックの地理的細分化など、適切な場所で使用できる一般的なファセットがあります。

Other tables are applied only to specific areas of the schedules. 
他のテーブルは、スケジュールの特定の領域にのみ適用されます。

Facets can be combined to create a complex subject statement. 
ファセットは組み合わせて複雑な主題文を作成することができます。

Daniel Joudrey and Arlene Taylor describe faceted classification using an analogy: "If one thinks of each of the faces of a cut and polished diamond as a facet of the whole, one can picture a classification notation that has small notations standing for subparts of the whole topic, which are pieced together to create a complete classification notation." 
ダニエル・ジャウドレイとアーリーン・テイラーは、ファセット型分類をアナロジーを用いて説明しています。「カットされて磨かれたダイヤモンドの各面を全体のファセットと考えるなら、全体のトピックの部分を表す小さな表記があり、それらが組み合わさって完全な分類表記を作成する様子を思い描くことができます。」

Faceted classifications exhibit many of the same problems as classifications based on a hierarchy. 
ファセット型分類は、階層に基づく分類と同様の多くの問題を示します。

In particular, some concepts could belong in more than one facet, so their placement in the classification may appear to be arbitrary to the classifier. 
特に、いくつかの概念は複数のファセットに属する可能性があるため、分類におけるその配置は分類者にとって恣意的に見えることがあります。

It also tends to result in a complex notation because each facet must be distinguishable as recorded. 
また、各ファセットが記録された通りに区別可能でなければならないため、複雑な表記になる傾向があります。



## Retrieval 検索

Search in systems with faceted classification can enable a user to navigate information along multiple paths corresponding to different orderings of the facets. 
ファセット分類を持つシステムでの検索は、ユーザが異なるファセットの順序に対応する複数のパスに沿って情報をナビゲートできるようにします。 
This contrasts with traditional taxonomies in which the hierarchy of categories is fixed and unchanging. 
これは、カテゴリの階層が固定されて変わらない従来の分類法とは対照的です。[7] 
It is also possible to use facets to filter search results to more quickly find desired results. 
また、ファセットを使用して検索結果をフィルタリングし、より迅速に望ましい結果を見つけることも可能です。



## Examples of faceted classifications 事例：ファセット分類の例

### Colon classification for library materials 図書資料のためのコロン分類

The Colon Classification developed by S. R. Ranganathan is an example of general faceted classification designed to be applied to all library materials. 
S. R. ランガナタンによって開発されたコロン分類は、すべての図書資料に適用されるように設計された一般的なファセット分類の一例です。

In the Colon Classification system, a book is assigned a set of values from each independent facet. 
コロン分類システムでは、書籍は各独立したファセットからの値のセットが割り当てられます。

This facet formula uses punctuation marks and symbols placed between the facets to connect them. 
このファセットの公式は、ファセット間を接続するために配置された句読点や記号を使用します。

Colon classification was named after its use of the colon as the primary symbol in its notation. 
コロン分類は、その表記における主要な記号としてコロンを使用していることにちなんで名付けられました。

Ranganathan stated that hierarchical classification schemes like the Dewey Decimal Classification (DDC) or the Library of Congress Subject Headings are too limiting and finite to use for modern classification and that many items can pertain information to more than one subject. 
ランガナタンは、デューイ十進分類法（DDC）やアメリカ議会図書館の主題見出しのような階層的分類スキームは、現代の分類に使用するには制限が多く有限であり、多くのアイテムが複数の主題に関連する情報を持つ可能性があると述べました。

He organized his classification scheme into 42 classes. 
彼は自らの分類スキームを42のクラスに整理しました。

Each class can be categorized according to particular characteristics, that he called facets. 
各クラスは、彼がファセットと呼んだ特定の特性に基づいて分類できます。

Ranganathan said that there are five fundamental categories that can be used to demonstrate the facets of a subject: personality, material, energy, space and time. 
ランガナタンは、主題のファセットを示すために使用できる5つの基本的なカテゴリがあると述べました：パーソナリティ、マテリアル、エネルギー、スペース、そして時間。

He called this the PMEST formula: 
彼はこれをPMEST公式と呼びました：

- Personality is the most specific or focal subject. 
- パーソナリティは最も特定的または焦点となる主題です。

- Matter is the substance, properties or materials of the subject. 
- マターは主題の物質、特性、または材料です。

- Energy includes the processes, operations and activities. 
- エネルギーはプロセス、操作、活動を含みます。

- Space relates to the geographic location of the subject. 
- スペースは主題の地理的位置に関連します。

- Time refers to the dates or seasons of the subject. 
- 時間は主題の日付や季節を指します。

### Universal Decimal Classification ユニバーサル十進分類法

Another example of a faceted classification scheme is the Universal Decimal Classification (UDC), a complex multilingual classification that can be used in all fields of knowledge. 
ファセット分類スキームの別の例は、ユニバーサル十進分類法（UDC）であり、すべての知識分野で使用できる複雑な多言語分類です。

The Universal Decimal Classification scheme was created at the end of the nineteenth century by Belgian bibliographers Paul Otlet and Henri la Fontaine. 
ユニバーサル十進分類法は、19世紀末にベルギーの著作家ポール・オトレとアンリ・ラ・フォンテーヌによって作成されました。

The goal of their system was to create an index that would be able to record knowledge even if it is stored in non-conventional ways including materials in notebooks and ephemera. 
彼らのシステムの目標は、ノートやエフェメラの資料を含む非従来型の方法で保存されている場合でも知識を記録できるインデックスを作成することでした。

They also wanted their index to organize material systematically instead of alphabetically. 
彼らはまた、インデックスが資料をアルファベット順ではなく体系的に整理することを望んでいました。

The UDC has an overall taxonomy of knowledge that is extended with a number of facets, such as language, form, place and time. 
UDCは、言語、形式、場所、時間などの多くのファセットで拡張された知識の全体的な分類法を持っています。

Each facet has its own symbol in the notation, such as: "=" for language; "-02" for materials, "[...]" for subordinate concepts. 
各ファセットには、表記において独自の記号があり、例えば、言語には "="、材料には "-02"、従属概念には "[...]" が使用されます。

### Faceted Classification for Occupational Safety and Health 労働安全衛生のためのファセット分類

D. J. Foskett, a member of the Classification Research Group in London, developed classification of occupational safety and health materials for the library of the International Labour Organization. 
ロンドンの分類研究グループのメンバーであるD. J. フォスケットは、国際労働機関の図書館のために労働安全衛生資料の分類を開発しました。

After a study of the literature in the field, he created the classification with the following facets: 
彼はこの分野の文献を研究した後、以下のファセットを持つ分類を作成しました：

- Facet A: Occupational Safety and Health: General 
- ファセットA: 労働安全衛生：一般

- Facet B: Special Classes of Workers, Industries 
- ファセットB: 特殊な労働者、産業のクラス

- Facet C: Sources of Hazards: Fire, Machinery, etc. 
- ファセットC: 危険の源：火災、機械など

- Facet D: Industrial Accidents and Diseases 
- ファセットD: 産業事故と疾病

- Facet E: Preventive Measures, Protection 
- ファセットE: 予防措置、保護

- Facet F: Organisation, Administration 
- ファセットF: 組織、管理

Notation was solely alphabetic, with the sub-facets organized hierarchically using extended codes, such as "g Industrial equipment and processes", "ge Machines". 
表記は完全にアルファベットであり、サブファセットは「g 工業機器とプロセス」、「ge 機械」などの拡張コードを使用して階層的に整理されました。

### Art and Architecture Thesaurus (AAT) 美術と建築の用語集（AAT）

While not strictly a classification system, the AAT uses facets similar to those of Ranganathan's Colon Classification: 
厳密には分類システムではありませんが、AATはランガナタンのコロン分類に似たファセットを使用しています：

- Associated Concepts (e.g., philosophy) 
- 関連概念（例：哲学）

- Physical Attributes 
- 物理的属性

- Styles and Periods 
- スタイルと時代

- Agents (People/Organizations) 
- エージェント（人々/組織）

- Activities (similar to Ranganathan's Energy) 
- 活動（ランガナタンのエネルギーに類似）

- Materials (similar to Ranganathan's Matter) 
- 材料（ランガナタンのマターに類似）

- Objects (similar to Ranganathan's Personality) 
- オブジェクト（ランガナタンのパーソナリティに類似）



## Comparison between faceted and single hierarchical classification
## ファセット分類と単一階層分類の比較

Hierarchical classification refers to the classification of objects using one single hierarchical taxonomy. 
階層分類とは、単一の階層的分類法を用いてオブジェクトを分類することを指します。

Faceted classification may actually employ hierarchy in one or more of its facets, but allows for the use of more than one taxonomy to classify objects. 
ファセット分類は、実際にはその一つ以上のファセットに階層を使用することがありますが、オブジェクトを分類するために複数の分類法を使用することを許可します。

- Faceted classification systems allow the assignment of multiple classifications to an object, and enable those classifications to be applied by searchers in multiple ways, rather than in a single, predetermined order. 
- ファセット分類システムは、オブジェクトに複数の分類を割り当てることを可能にし、検索者がそれらの分類を単一の事前に決められた順序ではなく、複数の方法で適用できるようにします。

Multiple facets may be used as a first step in a search process. 
複数のファセットは、検索プロセスの最初のステップとして使用されることがあります。

For example, one may start from language or subject. 
例えば、言語や主題から始めることができます。

- Hierarchical classification systems are developed classes that are subdivided from the most general subjects to the most specific. 
- 階層分類システムは、最も一般的な主題から最も特定的なものへと細分化されたクラスを開発します。

- Faceted classification systems allow for the combination of facets to filter the set of objects rapidly. 
- ファセット分類システムは、ファセットの組み合わせを許可し、オブジェクトのセットを迅速にフィルタリングします。

In addition, the facets can be used to address multiple classification criteria. 
さらに、ファセットは複数の分類基準に対処するために使用できます。

- A faceted system focuses on the important, essential or persistent characteristics of content objects, helping it to be useful for categorization of fine-grained rapidly changing repositories. 
- ファセットシステムは、コンテンツオブジェクトの重要で本質的または持続的な特性に焦点を当てており、細かく変化するリポジトリの分類に役立ちます。

- In faceted classification systems one does not have to know the name of the category into which an object is placed a priori. 
- ファセット分類システムでは、オブジェクトが配置されるカテゴリの名前を事前に知っている必要はありません。

A controlled vocabulary is presented with the number of documents matching each vocabulary term. 
制御語彙は、各語彙用語に一致する文書の数と共に提示されます。

- New facets may be created at any time without disruption of a single hierarchy or reorganizing other facets. 
- 新しいファセットは、単一の階層を中断したり、他のファセットを再編成したりすることなく、いつでも作成できます。

- Faceted classification systems make few assumptions about the scope and organization of the domain. 
- ファセット分類システムは、ドメインの範囲と組織についてほとんど仮定を行いません。

It is difficult to break a faceted classification schema. 
ファセット分類スキーマを壊すことは難しいです。
